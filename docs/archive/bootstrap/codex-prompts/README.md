# Archived Codex prompts

This folder keeps bootstrap and milestone build-history prompts out of `.github/codex/prompts/`.

Use `.github/codex/prompts/` only for:

- the active Codex workflow prompt
- real product-facing Codex prompt outputs written by the CLI

The files here are retained as maintainer history and reference material. Product-facing example prompts now live under `examples/demo/`.

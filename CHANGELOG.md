# Changelog

All notable changes to ContextForge will be documented in this file.

## Unreleased

- Release automation now retries npm version verification for several minutes after publish so registry propagation delays do not immediately fail the workflow.

## 0.1.0

- Local-first CLI published as `@xiwuqi/contextforge` for repository scan/init, task-pack compilation from markdown or GitHub issue sources, and compact exports for Codex, Claude Code, and Cursor.
- Stale-context linting plus deterministic fixture evals, packaged smoke validation, publish dry-run, and release-candidate checks.
- Checked-in demo assets, bilingual README onboarding, community contribution templates, and manual-dispatch release automation for public releases.
